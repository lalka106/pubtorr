# PubTorr — каталог парсеров для Lampa

Плагин для [Lampa](https://github.com/yumata/lampa): добавляет в настройках раздел **«Каталог парсеров»** и позволяет быстро выбрать публичный Jackett/Prowlarr‑парсер для торрентов без ручного ввода URL и ключа.

## Для чего нужен скрипт

В Lampa для поиска раздач через торренты нужен парсер (Jackett и аналоги). Обычно адрес и ключ вводят вручную. **PubTorr** предлагает готовый список публичных парсеров проекта LME — достаточно выбрать пункт из списка, и плагин сам пропишет `jackett_url`, `jackett_key` и тип парсера в хранилище Lampa.

### Возможности

- Каталог парсеров в **Настройки → Парсер**
- Проверка доступности серверов (доступен / ошибка ключа / недоступен)
- Сохранение выбранного парсера между запусками
- Удобный интерфейс для **TV‑пульта**: подсветка фокуса, цветные статусы, отметка выбранного парсера
- Совместимость со **старыми TV** (в т.ч. LG webOS 3.x): без `Object.values`, `Array.find`, `Promise.allSettled` и CSS‑переменных

## Установка

1. В Lampa откройте настройки плагинов и добавьте URL скрипта:

   ```
   https://lalka106.github.io/pubtorr/pubtorr.js
   ```

2. Включите **«Использовать парсер»** в настройках Lampa.
3. Откройте **Настройки → Парсер → Каталог парсеров**, выберите сервер и подтвердите **OK** на пульте.

Управление в каталоге: **↑ / ↓** — перемещение, **OK** — выбор, **Назад** — закрыть окно.

## Отказ от ответственности
